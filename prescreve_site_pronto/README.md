# Prescreve

Landing page institucional do Prescreve — telemedicina vertical especializada em renovação de receita continuada.

## Estrutura

- `index.html` — Landing page completa (single file, 137KB)
- `assets/prescreve_demo_web.mp4` — Vídeo demo otimizado (7MB)
- `assets/video_poster.jpg` — Poster do vídeo

## Deploy

Site estático puro. Compatível com:
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
- Qualquer host estático

Sem build step. Sem dependências. Apenas arquivos estáticos.

## Local

Abra `index.html` direto no navegador.

## Tecnologias

- HTML5 + CSS3 puro
- SVG inline (logo)
- Video HTML5 com autoplay muted loop
- Fonts via Google Fonts (Inter + Source Serif 4)
